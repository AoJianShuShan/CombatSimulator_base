# 基础战斗模拟器

一个零依赖的前端 TypeScript 战斗模拟器骨架，当前只实现最基础的 `攻击 - 防御` 回合制流程，但模型已经拆成：

- 领域模型：描述单位、战斗配置、事件与结果
- 模拟引擎：纯函数执行战斗，不依赖 UI
- Web UI：配置 N vs N 阵容、运行模拟、查看事件日志

## 运行

```bash
npm run build
npm run dev
```

默认会在 `http://localhost:4173` 启动静态服务。

## 当前规则

- 双方单位按速度从高到低行动
- 每次行动攻击当前第一个存活敌人
- 伤害公式：`max(最小伤害, 攻击 - 防御)`
- 任一方全灭或达到最大回合数后结束
- 输出完整事件流：战斗开始、回合开始、单位行动、伤害、死亡、战斗结束

## 目录

```text
src/
  domain/      领域模型
  simulator/   战斗引擎
  ui/          前端页面
scripts/       本地 build / dev / smoke 脚本
dist/          构建输出
```

## 后续扩展建议

- 单位参数：暴击、闪避、命中、技能、冷却、Buff / Debuff
- 战斗参数：随机种子、地图、阵型、行动条、目标选择策略
- 后端化：将 `src/domain` 与 `src/simulator` 提取为共享协议，前端只负责配置与可视化
